Main Concepts

The Web

The web is essentially a collection of documents. Most of these documents are HTML documents (but there are also PDFs, image files, etc). A user uses a web browser on his or her computer to view the Internet. In order to do this, the user's computer must send a request with HTTP (hypertext transfer protocol) to servers which store the HTML pages. The Internet is just a collection of many servers connected together, interpreted through a web browser.

Some definitions:

  • HTML(Hyper Text Markup Language) is the main type of document on the Web, and it is the language used to structure sites for the browsers,
  • HTTP(Hyper Text Transfer Protocol) is the main protocol of the web,
  • Servers are computers/machines that serve information to a client (usually a requester browser),
  • Clients and Browsers are usually software that request information on a server and display the response in a meaningful way to the user.

HTML file structure

As markup languages go, the most basic unit on HTML the tag. To define a tag a word is surrounded by angle brackets (eg. <word>). Most tags have an opening tag and a closing tag, but that is not mandatory and a closing tag has a forward slash after the first bracket (eg. </word>). Some tags have prorieties known as an attributes. For example, the anchor tag (<a>), used to create links (click here to go to Udacity website), has the href that indicates the hyperlink reference on the anchor for the user action.

Tags can have more than one attribute. Let's take the a tag, used for creating a linking must have a href attribute and also target attribute to change the behavior on how to get to the anchor reference.

Some tags do not need closing tags. The img tag is also known as a void tag because there is no content and therefore does not need closing tags, also is very common on newer HTML a notation as a self-closed with a slash at the end of the openning tag like: <img attributes />.

Display of elements

There are two main display intrinsic to the HTML elements inline and block display (also reffered as inline and block tags)

Inline tags are tags which occur within a text and do not disrupt the text structure. Many common tags are inline tags such as image: img, emphasis: em, bold: b and the span: span.

On the other hand, a block tag, is one that sections off an entire section of text, creaking an invisible "box" around an element. The most common block tags are the p and the div tags.

The difference between inline and block tags can be illustrated by comparing the span and div tags. Both tags are known as container tags as they surround an element to share attributes with children elements. However, span is an inline tag, existing within a line, while div is a block tag, creating a "box" or block around an element (udacity video on inline and block display).
